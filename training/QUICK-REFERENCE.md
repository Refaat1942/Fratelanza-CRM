## Daily tasks by role / المهام اليومية حسب الدور

### Cashier / صندوق

| Task | Go to | Steps |
|---|---|---|
| Check if drug is available | **Products** | Search name → check stock & price |
| Sell to customer | **Invoices** | Create → select client → add items → Record Payment |
| Print receipt | **Invoices** | Open invoice → Print |
| Send via WhatsApp | **Invoices** | Open invoice → WhatsApp button |
| New customer | **CRM** | Add Client → name + phone |

### Stock clerk / أمين مخزن

| Task | Go to | Steps |
|---|---|---|
| Check low stock | **Products** | Look at yellow/red badges at top |
| Receive delivery | **Purchase Orders** | Open PO → Receive |
| Manual stock change | **Products** | Adjust Stock → in/out + reason |
| Order from supplier | **Purchase Orders** | Create PO → select supplier + products |
| Check stock history | **Products** | History button on any product |

### Pharmacist / صيدلي

| Task | Go to | Steps |
|---|---|---|
| Look up drug name | **Medicine Master** | Search in Medical workspace |
| View prescription | **Prescriptions** | Search patient → view/print |
| Check sellable stock | **Products** | Search in General workspace |

### Manager / مدير

| Task | Go to | Steps |
|---|---|---|
| Today's revenue | **Dashboard** | Check KPI cards |
| Add staff user | **Settings** | Users → Add User → set role |
| Weekly report | **Reports** | Select date range |
| Add expense | **Finance** | Add Expense |

## Key rules / قواعد مهمة

| ✅ Do | ❌ Don't |
|---|---|
| Use **Adjust Stock** for every stock change | Edit stock number directly |
| Always select a **client** on invoices | Create invoice without customer |
| Click **Receive** when goods arrive | Forget to receive PO |
| Write a **reason** on stock adjustments | Leave reason blank |
| Log out on shared PCs | Share your password |
| Search before adding duplicate customer | Create same customer twice |
