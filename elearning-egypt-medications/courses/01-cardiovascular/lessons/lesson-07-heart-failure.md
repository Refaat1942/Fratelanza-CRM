## 1. Heart failure pharmacotherapy

### 1.1 HFrEF — four pillars of GDMT (ESC 2023)

| Pillar | Drug class | Egyptian examples |
|---|---|---|
| 1 | ACE-I/ARB/ARNI | Tritace, Tareg, Entresto (limited) |
| 2 | Beta-blocker | Concor, Dilatrend, Betaloc ZOK |
| 3 | MRA | Aldactone (spironolactone) |
| 4 | SGLT2 inhibitor | Forxiga (dapagliflozin), Jardiance (empagliflozin) |

**Evidence:** PARADIGM-HF (sacubitril/valsartan), DAPA-HF/EMPEROR-Reduced (SGLT2i) — mortality and hospitalization reduction.

### 1.2 Diuretics in HF

| Drug | Brand (Egypt) | Role |
|---|---|---|
| Furosemide | Lasix, EIPICO generic | Symptom relief — congestion |
| Torasemide | Demadex | Alternative loop |
| Spironolactone | Aldactone | Mortality benefit (RALES) + diuresis |
| Metolazone | Limited | Add-on for diuretic resistance |

### 1.3 Digoxin — Egyptian market

| Parameter | Details |
|---|---|
| **Brand** | Lanoxin, Digoxin (Memphis, EIPICO) |
| **Mechanism** | Na/K-ATPase inhibition → ↑ contractility; vagal tone → ↓ HR in AF |
| **Dosing** | 0.125–0.25mg OD; adjust for renal function and age |
| **Level** | Therapeutic 0.5–1.0 ng/mL (narrow therapeutic index) |
| **Approx. cost** | 40–100 EGP/month |

**Digoxin toxicity:** Nausea, visual changes (yellow halos), arrhythmias (any arrhythmia in patient on digoxin = toxicity until proven otherwise). Hypokalemia ↑ toxicity risk.

**Contraindications:** WPW with AF, hypertrophic cardiomyopathy, complete heart block.

## 2. Antiarrhythmics

### 2.1 Vaughan Williams classification

| Class | Mechanism | Egyptian examples |
|---|---|---|
| **Ia** | Na⁺ channel block (moderate) | Disopyramide (limited) |
| **Ib** | Na⁺ channel block (short) | Lidocaine (IV) |
| **Ic** | Na⁺ channel block (strong) | Propafenone (Rythmol — limited) |
| **II** | Beta-blockade | Concor, Inderal |
| **III** | K⁺ channel block | Cordarone (amiodarone), Sotalol |
| **IV** | Ca²⁺ channel block | Verapamil, Diltiazem |

### 2.2 Amiodarone — Egyptian market

| Parameter | Details |
|---|---|
| **Brand** | Cordarone (Sanofi), Amiodarone (Pharco, Memphis) |
| **Indications** | AF/flutter, VT/VF (refractory), rate/rhythm control |
| **Loading** | 200mg TDS × 1 week → 200mg BD × 1 week → 200mg OD maintenance |
| **Half-life** | 40–60 days — effects persist months after stopping |
| **Monitoring** | TFT (hypo/hyperthyroid), LFTs, pulmonary fibrosis, corneal deposits |
| **Interactions** | ↑ warfarin INR significantly; ↑ digoxin levels (↑ toxicity) |
| **Approx. cost** | 80–200 EGP/month |

### 2.3 Rate control in AF

| Agent | Brand (Egypt) | Notes |
|---|---|---|
| Metoprolol | Lopressor, Betaloc | First-line rate control |
| Bisoprolol | Concor | HF + AF |
| Diltiazem | Dilzem (limited) | Avoid in HFrEF |
| Digoxin | Lanoxin | Elderly, sedentary; not for exercise |
| Amiodarone | Cordarone | Rhythm + rate; reserve for refractory |

**Counseling script (Arabic):**

> "قصور القلب محتاج التزام بكل الأدوية — مثبط ACE، الكونكور، الألداكتون، وممكن فورزيجا. متوقفش المدرات لو تحسن عندك — الدكتور هينظم الجرعة. لو على لانوكسين: لو حسيت غثيان، رؤية ملونة، أو خفقان غريب، روح للطوارئ. كوردارون محتاج تحليل غدة وكبد كل 6 شهور."

## 3. Contraindications & interactions summary

| Drug | Critical interaction |
|---|---|
| Digoxin | Amiodarone ↑ digoxin levels 70% — reduce digoxin dose 50% |
| Amiodarone | Warfarin — reduce warfarin dose ~30–50% when starting |
| Beta-blocker + verapamil | Severe bradycardia |
| SGLT2i | DKA risk if insulin stopped abruptly; genital infections |
| NSAIDs | Fluid retention — worsen HF |

## 4. Pharmacy practice scenario

### Scenario 1.7 — HF patient wants to stop diuretic
> *Patient with HFrEF on Tritace + Concor + Aldactone + Lasix 40mg. Feeling well, wants to stop Lasix to reduce bathroom trips.*

**Response:**
1. Explain diuretic treats congestion symptoms — GDMT drugs (ACE-I, BB, MRA) treat mortality
2. Do NOT advise stopping without physician — risk of fluid overload
3. Suggest physician visit — may reduce to 20mg or alternate-day dosing
4. Counsel daily weight monitoring — gain >2kg in 3 days → call physician
5. Check salt intake counseling — Egyptian diet often high sodium

---

## Key takeaways

> 📌 **Key Takeaways — Lesson 1.7**
> - HFrEF GDMT: ACE-I/ARB + beta-blocker + MRA + SGLT2i — all four pillars
> - Lasix (furosemide) — symptom relief; Aldactone — mortality benefit
> - Lanoxin (digoxin) — narrow index; toxicity worsened by hypokalemia and amiodarone
> - Cordarone (amiodarone) — most used class III in Egypt; monitor thyroid, liver, lungs
> - Amiodarone dramatically increases warfarin effect — common Egyptian interaction
> - Forxiga/Jardiance now part of HF therapy regardless of diabetes status
