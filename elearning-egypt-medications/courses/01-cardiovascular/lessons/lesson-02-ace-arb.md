## 1. Mechanism of action

### 1.1 RAAS pathway

```
Renin (kidney) → Angiotensinogen → Angiotensin I
    → ACE (lung/vessels) → Angiotensin II
    → Vasoconstriction + Aldosterone release
    → Na+/water retention + BP elevation + cardiac remodeling
```

**ACE inhibitors** block angiotensin-converting enzyme → ↓ Angiotensin II → vasodilation, ↓ aldosterone.

**ARBs** block AT1 receptor → same downstream effect without affecting bradykinin (less cough).

### 1.2 Clinical effects (evidence)

| Outcome | Evidence | NNT (5 years) |
|---|---|---|
| BP reduction | ACE-I: SBP ↓ 8–10 mmHg | — |
| Heart failure mortality | SOLVD, CONSENSUS trials | 1 in 6 lives saved |
| Post-MI remodeling | SAVE, AIRE trials | Significant LV function preservation |
| Diabetic nephropathy | RENAAL (losartan) | Slows proteinuria progression |
| Stroke prevention | PROGRESS (perindopril) | 1 in 28 for recurrent stroke |

### 2.2 ACE inhibitor side effects

| Effect | Frequency | Management |
|---|---|---|
| Dry cough | 5–20% | Switch to ARB |
| Hyperkalemia | 3–5% | Monitor K+; avoid K+ supplements, K-sparing diuretics |
| Angioedema | 0.1–0.7% | **STOP immediately** — medical emergency |
| First-dose hypotension | Captopril > others | Take first dose at bedtime; start low dose |
| Renal function change | Expected initially | Monitor creatinine — acceptable rise <30% |
| Teratogenic (pregnancy) | Category D | **Contraindicated** — counsel women of childbearing age |

**Counseling script — ACE cough (Arabic):**
> "السعال الجاف اللي بدأ بعد الدواء ده ممكن يكون من المادة الفعالة. مش حساسية ومش خطير، بس مزعج. لو استمر، الدكتور ممكن يغيرك لدواء تاني من نفس المجموعة (زي تارج) من غير سعال."

### 3.2 ACE-I vs ARB — when to choose

| Situation | Choice |
|---|---|
| New HTN, no contraindication | ACE-I (cheaper generics in Egypt) |
| ACE cough develops | Switch to ARB (same class benefit) |
| Post-MI with LV dysfunction | ACE-I preferred (more evidence) |
| Diabetic nephropathy | Either — losartan/irbesartan strong evidence |
| Pregnancy | **Neither** — contraindicated |
| Hyperkalemia risk | Use cautiously; monitor K+ both classes |
| Cannot tolerate either | CCB or thiazide |

**NEVER combine ACE-I + ARB** — increased harm (hyperkalemia, renal failure) without benefit (ONTARGET trial).

## 4. Renal dosing

| eGFR (mL/min) | ACE-I/ARB action |
|---|---|
| >60 | Full dose |
| 30–60 | Start low; monitor creatinine in 1–2 weeks |
| <30 | Use cautiously; nephrology referral; some avoid |
| Dialysis | Losartan, telmisartan — some dialyzable |

**Rule:** Acceptable creatinine rise **<30%** from baseline after starting ACE-I/ARB. If >30%, investigate renal artery stenosis.
