## After downloading

1. Unzip the file
2. Open `READING-GUIDE.md` or `full-curriculum/COURSE-01-cardiovascular.md`
3. Read in any Markdown viewer, Word, or convert to PDF

### Convert to PDF
- Upload any `.md` file to https://www.markdowntopdf.com/
- Or use VS Code "Markdown PDF" extension
