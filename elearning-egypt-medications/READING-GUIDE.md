## Convert to PDF for printing

**Online:** Upload `COURSE-01-cardiovascular.md` to https://www.markdowntopdf.com/

**VS Code:** Install "Markdown PDF" extension → right-click file → Export PDF

**Command line:**
```bash
pandoc full-curriculum/COURSE-01-cardiovascular.md -o Course-01.pdf
```
