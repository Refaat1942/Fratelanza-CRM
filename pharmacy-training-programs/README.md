## How to deliver / كيفية التقديم

### Recommended format per module

1. **Pre-reading** (1 hour) — trainees read module document before class
2. **Lecture** (40% of module hours) — trainer presents scientific content
3. **Case study workshop** (30%) — small groups work through clinical scenarios
4. **Calculation lab** (15%) — dosing, dilution, infusion rate exercises
5. **Assessment** (15%) — MCQ + short answer, pass mark 70%

### Materials needed

- Printed or PDF module documents for each trainee
- Whiteboard for calculations
- Sample prescriptions (anonymized) for practical exercises
- BNF / drug reference book or equivalent app
- Calculator

### Certification

After completing all 10 modules with ≥70% on each assessment, issue internal certificate:
**"Certificate of Completion — Comprehensive Pharmacy Training Program (120 Hours)"**
