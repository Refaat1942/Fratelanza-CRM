## Section 5.1 — Interaction mechanisms

### 5.1.1 Pharmacokinetic (PK) interactions

| Mechanism | Effect | Classic example |
|---|---|---|
| **Absorption** | Chelation, pH change, P-gp efflux | Tetracycline + calcium/iron → reduced absorption |
| **Distribution** | Protein binding displacement | Warfarin + aspirin → transient free warfarin ↑ |
| **Metabolism (CYP inhibition)** | Substrate levels ↑ | Ketoconazole + simvastatin → rhabdomyolysis |
| **Metabolism (CYP induction)** | Substrate levels ↓ | Rifampicin + oral contraceptives → contraceptive failure |
| **Excretion** | Renal tubular competition | Probenecid + penicillin → penicillin levels ↑ (therapeutic use) |

**Grapefruit juice mechanism:** Furanocoumarins irreversibly inhibit intestinal CYP3A4. Effect lasts **24–72 hours**. One glass can increase substrate AUC by **300%+** (felodipine, simvastatin, cyclosporine).

### 5.1.2 Pharmacodynamic (PD) interactions

| Mechanism | Example | Clinical effect |
|---|---|---|
| Synergistic toxicity | Opioid + benzodiazepine | Respiratory depression (FDA black box warning) |
| Antagonistic effect | Beta-blocker + salbutamol | Reduced bronchodilation |
| Additive QT prolongation | Azithromycin + ondansetron + haloperidol | Torsades de pointes risk |
| Additive serotonergic | SSRI + tramadol + MAOI | Serotonin syndrome |
| Dual anticholinergic | Oxybutynin + chlorpheniramine | Confusion, urinary retention, constipation |

**Serotonin syndrome triad:** Mental status changes + autonomic hyperactivity + neuromuscular abnormalities (hyperreflexia, clonus). **Mortality if untreated.**

## Section 5.2 — Major interaction reference tables

### Table 5.1 — Anticoagulant interactions

| Interacting drug | Mechanism | Effect | Management |
|---|---|---|---|
| NSAIDs (ibuprofen, diclofenac) | PD + GI mucosa | Bleeding risk ↑↑ | Avoid combination; use paracetamol |
| Aspirin (low-dose) | PD additive | Bleeding ↑ | Often intentional; monitor INR |
| Azole antifungals | CYP2C9 inhibition | Warfarin levels ↑ | Reduce warfarin dose 25–50%; monitor INR |
| Rifampicin | CYP induction | Warfarin levels ↓ | Increase warfarin dose; close INR monitoring |
| Amiodarone | CYP2C9 inhibition | Warfarin levels ↑↑ | Reduce warfarin 30–50% |
| Broad-spectrum antibiotics | Gut flora ↓ → vit K ↓ | Warfarin effect ↑ | Monitor INR during and after course |
| St. John's Wort | CYP3A4, P-gp induction | Warfarin levels ↓ | Contraindicated combination |

### Table 5.2 — Cardiovascular interactions

| Combination | Risk | Evidence level |
|---|---|---|
| ACE inhibitor + K-sparing diuretic (spironolactone) | Hyperkalemia | RCT data — monitor K+ |
| ACE inhibitor + NSAID | AKI, reduced antihypertensive effect | Meta-analysis confirmed |
| Digoxin + amiodarone | Digoxin levels ↑ 70% | P-gp inhibition — halve digoxin dose |
| Digoxin + verapamil | Bradycardia, digoxin levels ↑ | Avoid if possible |
| Statins + gemfibrozil | Rhabdomyolysis | Gemfibrozil inhibits statin glucuronidation — contraindicated with simvastatin |
| Nitrates + PDE5 inhibitors (sildenafil) | Severe hypotension | **Contraindicated** — 24–48h washout |

### Table 5.3 — CNS interactions

| Combination | Risk | Mechanism |
|---|---|---|
| MAOI + SSRI | Serotonin syndrome — **contraindicated** | Serotonin accumulation; 2-week washout (5 weeks for fluoxetine) |
| Opioid + benzodiazepine | Respiratory depression — **FDA black box** | Synergistic CNS depression |
| Opioid + gabapentin/pregabalin | Respiratory depression ↑ | FDA 2019 warning |
| Alcohol + any CNS depressant | Additive sedation | Counsel absolute avoidance |
| Levodopa + antipsychotic | Antipsychotic blocks dopamine — antagonizes levodopa | Avoid typical antipsychotics |

## Section 5.3 — Drug-disease interactions

| Disease state | Avoid/use caution | Reason |
|---|---|---|
| **Asthma** | Non-selective beta-blockers (propranolol) | Bronchospasm |
| **Heart failure** | NSAIDs, thiazolidinediones, diltiazem | Fluid retention, negative inotropy |
| **Renal impairment (eGFR <30)** | Metformin, NSAIDs, aminoglycosides | Accumulation, toxicity |
| **Hepatic cirrhosis** | Hepatotoxic drugs, sedatives | Reduced metabolism |
| **Glaucoma (closed-angle)** | Anticholinergics, decongestants | Pupil dilation → acute attack |
| **BPH** | Anticholinergics, alpha-agonists | Urinary retention |
| **Gout** | Thiazide diuretics, low-dose aspirin | Uric acid ↑ |
| **Myasthenia gravis** | Aminoglycosides, fluoroquinolones, Mg | Neuromuscular blockade worsening |
| **Porphyria** | Barbiturates, carbamazepine, sulfonamides | Acute porphyria attack |

## Section 5.4 — Pharmacovigilance

### 5.4.1 ADR reporting science

**WHO definition of ADR:** *"A response to a drug which is noxious and unintended, and which occurs at doses normally used in man for prophylaxis, diagnosis, or therapy of disease."*

**Egyptian Pharmacovigilance Center (EPVC)** — Egyptian Drug Authority:
- Report suspected ADRs via EDA pharmacovigilance system
- **Who reports:** Pharmacists, physicians, patients
- **What to report:** Serious, unexpected, or suspected ADRs

**Serious ADR criteria (ICH E2A):**
- Death
- Life-threatening
- Hospitalization or prolongation
- Persistent disability
- Congenital anomaly
- Medically important event

### 5.4.2 Causality assessment — Naranjo scale

| Question | Yes | No | Unknown |
|---|---|---|---|
| Previous reports of this reaction? | +1 | 0 | 0 |
| ADR appeared after drug given? | +2 | -1 | 0 |
| Improved when drug stopped? | +1 | 0 | 0 |
| Reappeared on rechallenge? | +2 | -1 | 0 |
| Other causes possible? | -1 | +2 | 0 |
| Drug detected in toxic concentration? | +1 | 0 | 0 |
| Severity increased with dose? | +1 | 0 | 0 |
| Similar reaction to same drug class before? | +1 | 0 | 0 |
| ADR confirmed by objective evidence? | +1 | 0 | 0 |

**Score:** ≥9 = definite; 5–8 = probable; 1–4 = possible; 0 = doubtful

## Case study 5.1

> *Patient on phenytoin 300mg/day (therapeutic level), started fluconazole 200mg/day for fungal infection. 5 days later: ataxia, nystagmus, slurred speech.*

**Analysis:**
1. Fluconazole inhibits CYP2C9 and CYP2C19 → phenytoin metabolism ↓
2. Phenytoin levels may increase 50–100%
3. Toxicity symptoms match phenytoin overdose (nystagmus at levels >20 mcg/mL)
4. **Action:** Contact physician; check phenytoin level; likely reduce dose 25–50%; monitor
